| **Stats Bar** | Horizontal row of large numbers + labels with logos |
| **Footer** | Multi-column link grid, brand area, social icons, legal |
| **Grid Layout** | CSS Grid, responsive 1→2→3→4 columns |
| **Typography** | Large bold headings (h1-h3), clean sans-serif body, uppercase labels |
