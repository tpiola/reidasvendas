# 🏭 SOP — Produzir 50 sites/dia

## Arquitetura de time (mínimo viável para 50/dia)

> Com IA + templates, 1 builder entrega 10–15 sites/dia. 4 builders = 50/dia.

## Pipeline padrão de 1 site (ciclo 6h)

1. Brief (30min): formulário Typeform → Notion.
1. Pesquisa concorrência (30min): 5 concorrentes locais → insights.
1. Copy (45min): prompt padrão no Claude com brief + nicho.
1. Montagem (90min): template + seções + imagens (Unsplash/IA).
1. SEO local (30min): schema, GMB, títulos, descrições.
1. Automação (30min): WhatsApp + form + pixel.
1. QA (30min): checklist de 20 itens.
1. Deploy (15min): Vercel + domínio + SSL.
1. Entrega (30min): vídeo explicativo + handoff ao cliente.
## Stack oficial

* Builder: Framer (cliente edita depois) ou Next.js + Tailwind + shadcn (custom)
* Hospedagem: Vercel
* Domínio: Registro.br ou Cloudflare
* Formulários: Tally / Typeform
* WhatsApp: Chatwoot / Typebot / Z-API
* Analytics: Plausible / GA4 / Meta Pixel
* CRM: Notion (este) ou Pipefy
* Financeiro: Asaas / Stripe / Mercado Pago
* NF: Nota Controle / eNotas
* IA: Claude + GPT + Gemini + Cursor
